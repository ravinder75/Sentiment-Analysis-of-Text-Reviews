# Sentiment Analysis of Text Reviews

This project is an NLP-based Sentiment Analysis application that classifies text reviews as Positive or Negative using Machine Learning and Deep Learning techniques.

## Project Goals

- Learn Natural Language Processing (NLP)
- Perform text preprocessing
- Build machine learning models
- Create a prediction API
- Develop a simple web interface
- Deploy the application

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- NLTK
- Jupyter Notebook

## Dataset

IMDb Movie Reviews Dataset
